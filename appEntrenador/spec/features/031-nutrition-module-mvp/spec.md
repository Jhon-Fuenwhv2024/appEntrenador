# 031 · Módulo de Nutrición MVP (Macros y Calorías)

## Objetivo
El entrenador le asigna objetivos diarios al cliente: Calorías, Proteínas, Carbohidratos y Grasas. El cliente ve esto en su Dashboard.

## Requisitos
- Tabla `nutrition_plans` (client_id, trainer_id, calories, protein, carbs, fats).
- UI Entrenador: Formulario para asignar macros.
- UI Cliente: Tarjeta resumen en el dashboard.
