# 032 · Seguimiento de Hábitos Diarios (Opcional)

## Objetivo
Un sistema de "Checklist" diario. El entrenador configura hábitos personalizados para el cliente (ej. "Beber 2L de agua"). El cliente los marca como completados cada día. Es opcional por cliente.

## Requisitos
- Tablas `habits` y `habit_logs`.
- UI Entrenador: Asignar/activar hábitos a un cliente.
- UI Cliente: Checklist diario en el dashboard (solo si tiene hábitos).
