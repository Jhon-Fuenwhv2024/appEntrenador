# 033 · Fotos de Progreso y Check-in Semanal

## Objetivo
Subida de fotos mensuales para comparación visual y un cuestionario recurrente (sueño, fatiga, digestión) para el ajuste de rutinas.

## Requisitos
- Almacenamiento de imágenes (S3 o local/estático para MVP).
- Tabla `progress_photos` y `weekly_checkins`.
- UI Cliente: Formulario de check-in y subida de fotos.
- UI Entrenador: Revisión de check-ins y galería de fotos.
