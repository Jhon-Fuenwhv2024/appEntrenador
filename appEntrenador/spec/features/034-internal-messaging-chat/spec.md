# 034 · Mensajería Interna (Chat MVP)

## Objetivo
Un canal de comunicación directo y asíncrono entre el entrenador y sus alumnos, aislado por base de datos.

## Requisitos
- Tabla `messages` (sender_id, receiver_id, content, timestamp).
- Endpoints para enviar y recibir mensajes.
- UI: Interfaz de chat tipo WhatsApp.
