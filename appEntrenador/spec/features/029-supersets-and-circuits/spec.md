# 029 · Superseries y Circuitos

## Objetivo
Modificar la estructura de la base de datos para permitir agrupar 2 o más ejercicios (Ej. 1A y 1B). En el Workout Player, se muestran juntos para que el cliente los ejecute sin descanso entre ellos.

## Requisitos
- Modificar tabla `routine_exercises` para incluir `superset_group` o similar.
- Formulario de rutinas: UI para agrupar ejercicios.
- Workout Player: Renderizar ejercicios agrupados y manejar su progresión conjunta.
