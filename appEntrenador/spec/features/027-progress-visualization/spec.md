# 027 · Visualización de Progreso (Gráficas)

## Objetivo
Conectar una librería como Chart.js o ApexCharts para que el cliente vea una gráfica de líneas de su peso corporal, su IMC y la evolución del peso levantado en sus ejercicios clave a lo largo del tiempo.

## Requisitos
- Integración de librería de gráficos (Chart.js / ApexCharts).
- Endpoint para obtener datos históricos formateados para gráficas.
- UI: Pestaña o sección de gráficas en "Mi Progreso".
